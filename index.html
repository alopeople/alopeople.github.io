Cloud security means that the more users, the more secure each user, because such a large number of users, enough to cover every corner of the Internet, as long as a site is hacked or a new Trojan appeared, it will be immediately intercepted. Cloud security monitors software anomalies, obtains Internet viruses, pushes them to the server for analysis, and finally distributes solutions to each client. What is the likely impact?

One of the advantages of cloud service is cost saving. In a home enterprise, cloud service enables the company not to invest in similar physical resources. That is to say, the company does not need to spend money on its own server, and employees can use all the resources of the cloud to do more efficient work. The cloud features the ability to access another application over the network without actually downloading it, saving more disk space. On the other hand, when companies need more storage space, they only need to add a virtual server from the cloud, which is much easier than installing and configuring a server in real life. The downside of the cloud is the occasional loss of data. Of course, that's only in a few cases. Even so, most companies try to store key information themselves, rather than uploading it to the cloud.

How will this affect you?

For me, I am a person who likes technology and games. So I use cloud services a lot in real life. When my phone is full, I can upload resources to the cloud, freeing up more storage for my phone, rather than deleting them. Or when I get a new phone or tablet, I can use the cloud to transfer data more easily. In terms of games, I am looking forward to Google STADIA platform very much. As I said before, I no longer need to spend a lot of money to buy game equipment, but only need a better network. All in all, I think cloud greatly facilitates our life.
